# LunchMate Monorepo

## 구조

- **Frontend**: Next.js (TypeScript) - `apps/frontend`
- **Backend**: NestJS (TypeScript) - `apps/backend`

## 개발 환경

- 패키지 매니저: pnpm
- 모노레포 구조: pnpm workspace

## 초기 세팅 방법

1. `pnpm install`
2. 각 앱 디렉토리에서 개발 서버 실행

## 기타

- ESLint, Prettier 등은 추후 통합 설정 예정
